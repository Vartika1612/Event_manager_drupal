This repository contains a custom Drupal 10 module named **Event Manager**, developed as part of an academic assignment.

The module allows administrators to configure events and users to register for those events with proper validation, email notifications, and reporting features.

---

## Features

- Event configuration by admin
- Event registration form for users
- AJAX-based dependent dropdowns
- Registration window validation
- Duplicate registration prevention
- Email notifications using Drupal Mail API
- Admin reporting dashboard
- CSV export of registrations
- Custom permission-based access control
